Archetypes

Archetyped ECS

This is an Entity-Component-System (ECS) architecture I've designed and implemented myself based on everything I've learned about how operating systems access memory and execute instructions. It follows an "archetype" based approach which makes iterating over many similar objects highly cache efficient.

Lighting

Physically Based Rendering

This a renderer that follows many modern rendering techniques. It supports rigid shadow mapping as well as soft shadowing. It also uses Physically Based Rendering (PBR) for realistic materials, and Image Based Lighting (IBL) for realistic ambient light. The renderer also has support for both point light and directional lights. My primary optimizations that make this possible are deffered rendering, as well as uniform batching and bindless textures. Point lights also use a local lights optimization, only rendering nearby fragments.
