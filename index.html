Archetypes

Geppetto Engine Main

This is a personal project of mine that is meant to be a playground where I can implement anything I think sounds cool. It's designed to be highly modular, so a new system or new component can be registered and start running with a single line of code. Many features get generated through reflection, such as UI and serialization.

Archetypes

Archetyped ECS

This is an Entity-Component-System (ECS) architecture I've designed and implemented myself based on everything I've learned about how operating systems access memory and execute instructions. It follows an "archetype" based approach which makes iterating over many similar objects highly cache efficient.

Lighting

Physically Based Rendering

This a renderer that follows many modern rendering techniques. It supports rigid shadow mapping as well as soft shadowing. It also uses Physically Based Rendering (PBR) for realistic materials, and Image Based Lighting (IBL) for realistic ambient light. The renderer also has support for both point light and directional lights. My primary optimizations that make this possible are deffered rendering, as well as uniform batching and bindless textures. Point lights also use a local lights optimization, only rendering nearby fragments.

Animations

Animations

These are animations and models that are loaded via Assimp. They are fully incorporated into the main renderer, and support path following and inverse kinematics. The engine also supports spring physics connections.

KnightLight Editor Gameplay

KnightLight

This is a project I worked on my softmore year of college as the technical lead of a team of 7. I developed the core of the project using a hierarchical object oriented design, where entities contained lists of components defining their behavior. This project was also my first time making a sophisticated serialization system. It had various features such as cacheable scenes and subscenes, as well engine state captures used for having multiple save files. All serialization was done utilizing RapidJSON.
